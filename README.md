## Overview

This repository contains the **Cactus Canyon Remake Pinball Guide**, a comprehensive, illustrated resource for fans and players of the Cactus Canyon Remake pinball project. It is designed to help players understand the game mechanics, bonus objectives, and optional modifications step by step.

The guide features:

- Detailed tables of all game phases and objectives
- Visual aids and images for quick understanding
- Step-by-step sequences to complete modes and challenges
- Optional mods to enhance gameplay and visual experience
- Structured layout for both web and printable versions

---

## Features

- **Rules & Objectives:** Clear, structured presentation of all pinball rules and objectives.
- **Bonus Scoring:** Tables show how to earn extra points and complete special objectives.
- **Optional Mods:** Links to downloadable mods with descriptions and images.
- **Print-Friendly:** Custom CSS ensures tables and images print cleanly without splitting content.
- **Future Upgrade:** Designed to integrate additional rule sets like the *Lyman Sheets Complete Upgrade*.
